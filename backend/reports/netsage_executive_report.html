Operational Impact: OSPF neighbor state cannot reach FULL. Routers drop OSPF Hello packets due to invalid signature, stopping all routing propagation between Head Office and Branch networks.

Core Fix: Align the OSPF message-digest authentication key on both connecting interfaces to use identical passwords.

  2. Select interface: 'interface GigabitEthernet0/0'.
